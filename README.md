# RICASE
Это лёгкий оверлей для стримов на Twitch, который позволяет зрителям открывать виртуальные кейсы
## Возможности

- Команда `!open` в чате → запуск анимации открытия кейса
- Разные редкости с настраиваемыми шансами
- Кулдаун **1 час** час на одного зрителя
- Предметы, которые уже были у пользователя, помечаются
- Автоматическая отправка результата в чат (при наличии OAuth-токена)
- Полностью настраиваемый список предметов и картинок

## Установка и запуск (3 минуты)

1. Скачай и распакуй архив с [последним релизом](https://github.com/RIPPER666YT/RICASE/releases)
2. Запусти файл `RICASE.exe`  
3. В открывшемся окне:

   **Вкладка «Основное»**
   - Twitch канал → пиши **без @ и без #** (пример: `ripper_cmertanoc`)
   - OAuth токен → вставь полностью (начинается с `oauth:`)  
   - Галочка «Открывать браузер при запуске» — можно оставить

4. Нажми **Сохранить настройки**
5. Нажми **Запустить сервер**

## Подключение к OBS (самый важный шаг!)

> **Важно:** ссылку http://127.0.0.1:5000 **нельзя** открывать в обычном браузере одновременно с OBS — иначе кейсы будут дублироваться!

1. В OBS → Источники → **+** → **Браузер**
2. URL: http://127.0.0.1:5000
3. **Ширина** и **Высота** → размер твоего экрана в OBS
4. Остальное по умолчанию
5. OK → правой кнопкой на источник → **Обновить**

Через 3–7 секунд должен появиться текст **ПОДКЛЮЧЕНО** зелёным цветом.

## Быстрая проверка

Напиши в свой чат: !open

Если всё правильно — через пару секунд запустится анимация → в чате появится результат (при наличии токена).

Если работает что то не так можно написать в тгк https://t.me/ripper_cmertanoc в сообщение каналу.
